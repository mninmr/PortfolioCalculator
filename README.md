# PortfolioCalculator

The forecasting engine is an interactive program that simulates the behavior of a portfolio of 
fixed rate mortgages under different macroeconomic scenarios. 

## Dependencies

* NumPy 
* Pandas
* PySide

https://pypi.python.org/pypi/PySide
